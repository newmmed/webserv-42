Autoindex

This server exposes an autoindex at /upload/. Use that link to browse files.

Files are saved under www/uploads using a timestamp-based name.

Delete a file

Copy a filename from the autoindex and run:

curl -X DELETE http://127.0.0.1:9900/upload/FILENAME

On success you'll get 200 and "Deleted"; unknown files return 404 with an error page.
