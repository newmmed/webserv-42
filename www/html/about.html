Overview

This is a minimalist HTTP server written in C++98 using non-blocking sockets and a single poll() loop as required by the project subject.

  • HTTP/1.0 & HTTP/1.1 request parsing
  • Persistent connections (keep-alive)
  • GET, POST, DELETE implementations
  • Location routing, redirects, error pages
  • Autoindex & file uploads with size limits

Testing

See scripts/test-curl.sh for automated curl-based tests.
